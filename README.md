# progressbar

Show a CLI progress bar for long running scripts

## Installation

with [basher](https://github.com/basherpm/basher)

	$ basher install pforret/progressbar

or with `git`

	$ git clone https://github.com/pforret/progressbar.git
	$ cd progressbar

## Usage

	> progressbar.sh [action]
